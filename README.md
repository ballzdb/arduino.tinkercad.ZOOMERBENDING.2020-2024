# Arduino Tinkercad ZOOMERBENDING Project (2020-2024)

## Overview

This repository contains Arduino and Tinkercad projects related to ZOOMERBENDING, spanning from 2020 to 2024. The project showcases a collection of circuit designs, code examples, and hardware prototypes developed over this period.

## Project Description

ZOOMERBENDING is an innovative Arduino-based project that combines digital electronics, circuit design, and programming. This archive documents the evolution and various iterations of the project from 2020 through 2024.

## Getting Started

### Prerequisites
- Arduino IDE installed
- Tinkercad account (for circuit simulations)
- Arduino-compatible microcontroller board
- Basic knowledge of electronics and programming

### Usage

1. **For Hardware Projects:**
   - Review the circuit schematics in the `schematics/` directory
   - Load the corresponding Arduino sketch from `sketches/` into the Arduino IDE
   - Upload the code to your Arduino board
   - Connect components according to the schematic

2. **For Tinkercad Simulations:**
   - Visit [Tinkercad](https://www.tinkercad.com)
   - Import or recreate circuits from the `circuits/` directory
   - Run simulations directly in Tinkercad
